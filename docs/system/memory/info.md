# info

### 操作概述

查看系统内存详细信息

### 命令参数

无参数

### 使用示例

```bash
acli system memory info
```

### 结果示例

```bash
MemTotal:       40679596 kB
MemFree:        20305748 kB
MemAvailable:   24797012 kB
Buffers:          912428 kB
Cached:          5285744 kB
SwapCached:            0 kB
Active:         12311456 kB
```
