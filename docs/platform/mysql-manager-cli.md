# mysql-manager-cli

### 操作概述

mysql管理工具

### 命令参数

无参数

### 使用示例

```bash
acli platform mysql-manager-cli --help
```

### 结果示例

```bash
NAME:
   mysql-manager-cli - Tools of mysql-manager

USAGE:
   mysql-manager-cli [global options] command [command options] [arguments...]

COMMANDS:
   debug           mysql manger debug tools
   status          Manager mysql service status
   master          Manage master db.
   replica         Manage replica db.
   service         Manage mysql service.
   mysql-managerd  Manage mysql-managerd service
   help, h         Shows a list of commands or help for one command

   cluster:
     cluster  Get cluster information

GLOBAL OPTIONS:
   --help, -h  show help
```
