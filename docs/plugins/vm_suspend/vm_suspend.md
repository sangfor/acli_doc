# vm_suspend

### 操作概述

虚拟机挂起检测

### 命令参数

```bash
-i|--vmid=string              （必要参数），虚拟机 vmid，示例：6025407369236
-t|--time=string              检查日期，默认今天，格式：YYYY-MM-DD，示例：2026-01-01
-v|--version                  显示当前插件的版本信息
-h|--help                     显示当前命令的帮助信息
```

### 使用示例

```bash
# 示例1：检测指定虚拟机的挂起问题
acli vm_suspend -i 6025407369236

# 示例2：检测指定虚拟机的挂起问题（指定日期）
acli vm_suspend -i 6025407369236 -t 2026-01-01

# 示例3：显示插件版本信息
acli vm_suspend -v

# 示例4：显示帮助信息
acli vm_suspend --help
```
