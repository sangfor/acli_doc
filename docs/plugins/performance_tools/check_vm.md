# check_vm

### 操作概述

检查虚拟机的状态（虚拟机配置文件、IO、MEM、CPU），同时自动检测虚拟机运行主机，必须输入虚拟机ID

### 命令参数

```bash
[vmid]                        （必填参数）虚拟机id，示例：6025407369236
-a|--checkall                 对告警等级为低的检测项也进行告警，默认对告警等级为低的检测项不进行告警
--checkdbvm                   检查数据库虚拟机配置 POC 测试的最佳实践
-d|--date=string              指定检查时间，仅部分检测项支持，格式：2006-01-02 15:04:05
-v|--desc                     打印虚拟机检测详细信息
--html-output=string          HTML 格式输出文件路径，不存在会自动创建（默认：/sf/data/local/performance_tools/check_result/checkvm_output.html）
-o|--output=string            输出文件路径，不存在会自动创建（默认：/sf/data/local/performance_tools/check_result/output.txt）
--lang=string                 设置命令行输出语言，可选项：zh_CN、en_US（全局参数）
-h|--help                     显示帮助信息
```

### 使用示例

```bash
# 示例1：检查指定虚拟机
acli performance_tools check_vm 1649206016202

# 示例2：检查指定虚拟机并打印详细信息
acli performance_tools check_vm 1649206016202 -v

# 示例3：检查指定虚拟机并指定检查时间
acli performance_tools check_vm 1649206016202 -d "2025-06-25 10:30:00"
```
