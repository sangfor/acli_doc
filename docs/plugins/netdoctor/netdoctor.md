# netdoctor

### 操作概述

进行网络相关问题的排障

### 命令参数

```bash
-c|--scene=string                （必要参数），排障场景选择，使用英文逗号拼接指定的场景，示例：vm
                                 可供选择的场景，枚举值：
                                    - vm（虚拟机丢包诊断）           - mgmt（管理网丢包诊断）
                                    - vxlan（VxLAN网丢包诊断）      - vs（存储网丢包诊断）
                                    - offline（主机离线诊断）        - perf（虚拟机时延卡顿诊断）
                                    - drop（物理口丢包/错包诊断）     - nic（主机网卡诊断）
                                    - upg（升级诊断）               - srv（虚拟网络服务异常诊断）
                                    - all（全场景诊断）
-d|--date=string                 指定日期的日志，示例：2023-06-25,07:37（日期和时间使用英文逗号分隔）
```

### 使用示例

```bash
# 示例1：默认不带参数，展示各排障场景供选择
acli netdoctor

# 示例2：指定场景进行诊断
acli netdoctor -c vm

# 示例3：指定场景并搜索指定日期的日志
acli netdoctor -c all -d 2023-06-26,07:37
```
