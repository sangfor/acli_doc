# restart

### 操作概述

重启mysql服务

### 命令参数

无参数

### 使用示例

```bash
acli service asv mysql restart
```

### 结果示例
```bash
acli service asv mysql restart

Shutting down MySQL (Percona Server)..............
Starting MySQL (Percona Cluster).....
{"data": null, "success": 1}
{"data": null, "success": 1}
```
