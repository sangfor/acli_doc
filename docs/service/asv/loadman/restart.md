# restart
操作概述: 重启loadman服务

命令参数:
```bash
无
```

使用示例:
```bash
acli service asv loadman restart
```

结果示例:
```bash
[ ok ] Stoping loadman: :[....] checking stop status: getcmdpid return 1, get pids: :  OK!.
[....] checking before start: getcmdpid return 1, get pids: :[....] Start (supervisor --lock-file '/var/run/supervisor/loadman.suplock' --f[ ok ile '*' --flag 011 -w '/sf/bin/loadman -f /sf/etc/loadman/loadman.yaml ') OK! 1533778.
```
