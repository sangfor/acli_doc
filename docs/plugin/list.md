# list

### 操作概述

获取已安装的插件列表

### 命令参数

无（该命令不需要参数）

### 使用示例

```bash
acli plugin list
```

### 结果示例

```bash
acli plugin list
    Name             |Source|Status|Version
    -----------------|------|------|-------
1   asan_ops         |官方包|开启  |1.0.0
2   netdoctor        |官方包|开启  |1.0.0
3   performance_tools|官方包|开启  |1.0.9
4   vm_start         |官方包|开启  |1.0.0
5   vm_suspend       |官方包|开启  |1.0.0

```
