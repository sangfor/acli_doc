# list

### Overview

Query the cluster VXLAN information list

### Command Parameters

No parameters

### Usage Example

```bash
acli network vxlan list
```

### Output Example

```bash
   vxlan_clusters:
      probe_enable: 1
      probe_latency_us: 10000
      probe_timeout_ms: 3000
      id: 8a5987e4-ff3e-47d0-8fd1-b36d2d566439
      probe_interval_s: 30
      jumbo_frame: 1
      port: 8472
```
