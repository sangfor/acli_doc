# delete

### Overview

Delete traffic mirroring policy

### Command Parameters
```bash
-i|--id=string    Required parameter, Traffic mirroring policy ID, Example: 3e1be084-e194-417e-a30d-652c3b415143
```

### Usage Example

```bash
acli network anet mirror delete --id c8e0562c-663d-41d6-be8c-4e6bb2617ae5
```

### Output Example

```bash
Command executed successfully.
```
