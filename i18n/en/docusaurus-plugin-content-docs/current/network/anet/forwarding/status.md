# status

### Overview

Query the forwarding core utilization of the current host

### Command Parameters

```bash
No parameters
```

### Usage Example

```bash
acli network anet forwarding status
```

### Output Example

```bash
core-1: 0%, busy: 3619350, idle: 2353555650
core-2: 0%, busy: 1718544, idle: 2351376870
core-3: 0%, busy: 2955078, idle: 2367196524
core-4: 0%, busy: 1622007, idle: 2355706239
```
