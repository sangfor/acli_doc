# migrate

### Overview

Migrate a virtual router

### Command Parameters
```bash
-r|--router-id=string    Required parameter, Router ID, Example: 3e1be084-e194-417e-a30d-652c3b415143
-h|--host-id=string      Required parameter, Node ID, Example: host-005056b2a30a
```

### Usage Example

```bash
acli network anet vrouter migrate --router-id 2df9fc9e-c067-4d25-b6d6-5c27a10ecff2 --host-id host-005056b2ec8d
```

### Output Example

```bash
Command executed successfully.
```
