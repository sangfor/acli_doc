# list

### Overview

Get the list of installed plugins

### Command Parameters

None (this command does not require parameters)

### Usage Example

```bash
acli plugin list
```

### Output Example

```bash
acli plugin list
    Name             |Source  |Status|Version
    -----------------|--------|------|-------
1   asan_ops         |Official|On    |1.0.0
2   netdoctor        |Official|On    |1.0.0
3   performance_tools|Official|On    |1.0.9
4   vm_start         |Official|On    |1.0.0
5   vm_suspend       |Official|On    |1.0.0

```
