# cleanup

### Overview

Clean up the directory and kill related processes for the specified plugin

### Command Parameters

```bash
-n|--name=string    Required parameter,Plugin name, only supports a single plugin
```

### Usage Example

```bash
acli plugin cleanup -n vm_start
```

### Output Example

```bash
acli plugin cleanup -n vm_start
Cleanup successful
```
