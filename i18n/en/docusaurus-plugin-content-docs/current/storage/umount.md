# umount

### Overview

Unmount the mount point of the specified datastore

### Command Parameters
```bash
-s|--storage-id=string    Required parameter, Specify a storage ID, Example: 36000c29a3c9419c600cdb98b5f823744_005056893c9d
```

### Usage Example

```bash
acli storage umount -s 36000c29a3c9419c600cdb98b5f823744_005056893c9d
```

### Output Example

```bash
Command executed successfully.
```
