# set

### Overview

Set the config of datareport
  
### Command Parameters

```bash
  -s|--storage-id=string              Required parameter,Specify a storage ID. Example: 36000c29a3c9419c600cdb98b5f823744_005056893c9d
  -f|--force=integer                  Forced modification (ignoring storage error), example: 1 forced, 0 not forced
```
  
### Usage Example

```bash
acli platform datareport set -s 019de1b3f4507b6991e610e0ed77c9df -f 1
```
  
### Output Example

```bash
host-0050568eb53e:2422079:1779351636:6928791085320
```
