# list

### Overview

Display GPU configuration information list

### Command Parameters

No parameters

### Usage Example

```bash
acli hardware gpu config list
```

### Output Example

```bash
# Currently only retrieves from the /sf/cfg directory
# gpu_info.ini and {hostname}_vgpu_load_file.json
gpu_info.ini
host-005056893c9d_vgpu_load_file.json
```
