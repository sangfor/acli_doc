# status

### Overview

View the vn-manager-service-api service status

### Command Parameters

No parameters

### Usage Example

```bash
acli service anet vn-manager-service-api status
```

### Output Example

```bash
vn-manager-service-api is started and running: 19271 19284 20229 20230 20232 20234 20235
```
