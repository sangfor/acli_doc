# status

### Overview

Obtain the RPS status on the node

### Command Parameters

No parameters

### Usage Example

```bash
acli service anet vn-cluster-service-api rps status
```

### Output Example

```bash
[0]
  hostId: host-002987263c8g
  isEnabled: 1
[1]
  hostId: host-761846915f3m
  isEnabled: 0
```
