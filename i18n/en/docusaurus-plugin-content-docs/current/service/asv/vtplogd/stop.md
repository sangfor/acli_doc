# stop

### Overview

Stop vtplogd service

### Command Parameters

No parameters

### Usage Example

```bash
acli service asv vtplogd stop
```

### Output Example
```bash
acli service asv vtplogd stop

Stoping vtplogd: :checking stop status: getcmdpid return 1, get pids: :  OK!.
```
