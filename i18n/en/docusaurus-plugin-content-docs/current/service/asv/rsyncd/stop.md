# stop

### Overview

Stop rsyncd service

### Command Parameters

No parameters

### Usage Example

```bash
acli service asv rsyncd stop
```

### Output Example
```bash
acli service asv rsyncd stop

Stoping rsyncd: :checking stop status: getcmdpid return 0, get pids: 4126757: .checking stop status: getcmdpid return 1, get pids: :  OK!.
```
