# restart

### Overview

Restart the MySQL service

### Command Parameters

No parameters

### Usage Example

```bash
acli service asv mysql restart
```

### Output Example
```bash
acli service asv mysql restart

Shutting down MySQL (Percona Server)..............
Starting MySQL (Percona Cluster).....
{"data": null, "success": 1}
{"data": null, "success": 1}
```
