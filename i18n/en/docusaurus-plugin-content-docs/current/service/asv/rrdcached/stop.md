# stop

### Overview

Stop rrdcached service

### Command Parameters

No parameters

### Usage Example

```bash
acli service asv rrdcached stop
```

### Output Example
```bash
acli service asv rrdcached stop

Stoping rrdcached: :checking stop status: getcmdpid return 0, get pids: 4125899: .checking stop status: getcmdpid return 1, get pids: :  OK!.
```
