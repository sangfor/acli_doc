# stop

### Overview

Stop the sangfor_waf service

### Command Parameters

No parameters

### Usage Example

```bash
acli service asv sangfor_waf stop
```

### Output Example
```bash
acli service asv sangfor_waf stop

Stoping sangfor_waf: :checking stop status: getcmdpid return 1, get pids: :  OK!.
```
