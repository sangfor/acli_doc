# status

### Overview

View the vtpdaemon service status

### Command Parameters

No parameters

### Usage Example

```bash
acli service asv vtpdaemon status
```

### Output Example
```bash
acli service asv vtpdaemon status

vtpdaemon is started and running: 4130670 4130715 4130720 4131165 4132427 4133450 4133945
```
