# get

### Overview

Get process op queue config

### Command Parameters
```bash
No parameters
```

### Usage Example

```bash
acli service asv vtpdaemon opqueue config get
```

### Output Example

```bash

process_op_queue_max_waiting: 999
process_op_queue_max_working: 14
```
