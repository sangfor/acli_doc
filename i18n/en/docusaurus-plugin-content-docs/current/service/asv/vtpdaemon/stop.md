# stop

### Overview

Stop vtpdaemon service

### Command Parameters

No parameters

### Usage Example

```bash
acli service asv vtpdaemon stop
```

### Output Example
```bash
acli service asv vtpdaemon stop

Stoping vtpdaemon: :checking stop status: getcmdpid return 1, get pids: :  OK!.
```
