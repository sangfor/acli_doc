# sync

### Overview

Synchronize the acli command to other nodes.

### Command Parameters
```bash
-o|--host=string    Target host IP, not specified means sync to all other nodes, example: 192,168.1.10
```

### Usage Example

```bash
acli plugin sync -o 192.168.1.10

acli plugin sync
```

### Output Example

```bash
acli command sync completed.
```
