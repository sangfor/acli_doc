# get

### Overview

Get virtual machine status

### Command Parameters
```bash
-v|--vm-id=string           Required parameter, VM ID, Example: 2848367193475
```

### Usage Example

```bash
acli vm status get -v 1072335336316
```

### Output Example

```bash
power: 1
status: 0
updated_at: 07:16:21
vmid: 1072335336316
```
