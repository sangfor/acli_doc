# get

### Overview

Get virtual machine disk path

### Command Parameters
```bash
-v|--vm-id=string           Required parameter, VM ID, Example: 2848367193475
```

### Usage Example

```bash
acli vm disk path get -v 1072335336316
```

### Output Example

```bash
/sf/data/a4ef106e_vs_vol_rep2/images/cluster/1072335336316.vm
```
