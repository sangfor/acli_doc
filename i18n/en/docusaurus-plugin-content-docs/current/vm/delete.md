# delete

### Overview

Delete VM

### Command Parameters
```bash
-v|--vm-id=string           Required parameter, VM ID, Example: 4872898137715
```

### Usage Example

```bash
acli vm delete -v 1072335336316
```

### Output Example

```bash
host-0050568e723c:1797593:1775785841:3009749796815
```
