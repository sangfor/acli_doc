# int3

### Overview

Dump guest memory using the INT3 instruction

### Command Parameters
```bash
-v|--vm-id=string           Required parameter, VM ID, Example: 4872898137715
```

### Usage Example

```bash
acli vm int3 -v 1072335336316
```

### Output Example

```bash
Successfully dumped virtual machine memory using int3 method!
```
