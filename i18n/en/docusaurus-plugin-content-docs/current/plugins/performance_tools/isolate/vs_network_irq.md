# vs_network_irq

### Overview

Automatically find storage communication port network card interrupt isolation

### Command Parameters

No parameters

### Usage Example

```bash
acli performance_tools isolate vs_network_irq
```

### Output Example

```bash
Storage port network interrupt core isolation failed: Current HCI version does not support setting storage port network interrupt cores, or current host has no VS
```
