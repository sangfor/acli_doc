# status

### Overview

Get monitoring data status

### Command Parameters

```bash
No parameters (this command does not require parameters)
```

### Usage Example

```bash
# Example 1: Get monitoring data status
acli performance_tools monitor status
```

### Output Example

```bash
# Monitoring exists
Monitoring program is running, pid: 1438433

# Monitoring does not exist
Monitoring program is not running
```
