# clean

### Overview

Delete monitoring data generated last time

### Command Parameters

```bash
No parameters (this command does not require parameters)
```

### Usage Example

```bash
# Example 1: Delete monitoring data generated last time
acli performance_tools monitor clean
```

### Output Example

```bash
Clean monitoring data files, are you sure to continue? (yes/no): yes
Monitoring data has been completely deleted
```
