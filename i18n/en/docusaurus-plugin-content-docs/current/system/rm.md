# rm

### Overview

Delete only files in the specified directory

### Command Parameters

No parameters
### Usage Example

```bash
acli system rm
```

### Output Example

```bash
Please use absolute paths. Files can only be deleted from the following directories:
/sf/cfg/
/sf/vs/vscfg/
/sf/data/
/run/
/var/
/sf/log/
/tmp/
/dev/
```
