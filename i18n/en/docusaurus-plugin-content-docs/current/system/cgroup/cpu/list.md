# list

### Overview

List all files in /cgroup/cpu

### Command Parameters

No parameters
### Usage Example

```bash
acli system cgroup cpu list
```

### Output Example

```bash

cgroup.clone_children
cgroup.procs
cgroup.sane_behavior
compute

cpuacct.usage_all
cpuacct.usage_percpu
cpuacct.usage_percpu_sys
cpuacct.usage_percpu_user
cpuacct.usage_sys
cpuacct.usage_user
cpu.cfs_period_us
cpu.cfs_quota_us
```
