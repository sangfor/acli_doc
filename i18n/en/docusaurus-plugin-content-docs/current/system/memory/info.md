# info

### Overview

View detailed system memory information

### Command Parameters

No parameters
### Usage Example

```bash
acli system memory info
```

### Output Example

```bash
MemTotal:       40679596 kB
MemFree:        20305748 kB
MemAvailable:   24797012 kB
Buffers:          912428 kB
Cached:          5285744 kB
SwapCached:            0 kB
Active:         12311456 kB
```
