# clear

### Overview

Clear multipath policy

### Command Parameters

```bash
-w|--wwid=string    Required parameter, logical storage ID, example: 3648bd4a100fa95b6550a2d9c000002ac
```

### Usage Example

```bash
acli system multipathconfig clear -w 3648bd4a100fa95b6550a2d9c000002ac
```

### Output Example

```bash
OK
```
