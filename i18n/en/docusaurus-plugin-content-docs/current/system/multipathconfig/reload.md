# reload

### Overview

Rescan disks to apply multipath configuration, default FC

### Command Parameters

```bash
-d|--disktype=string    Optional parameter, disk type, default is fc
```

### Usage Example

```bash
acli system multipathconfig reload
```

### Output Example

```bash
UPID:host-005056b22c8d:0027794D:BD6D5B:6A18ED6F:扫描FC磁盘::admin@vtp:
```
